# C Programming

A collection of C programming programs covering basic to intermediate concepts. This repository is created to improve problem-solving skills and strengthen programming fundamentals.

## 📌 Topics Covered

- Basic C Programs
- Input and Output
- Variables and Data Types
- Operators
- Conditional Statements (if, if-else, switch)
- Loops (for, while, do-while)
- Functions
- Arrays
- Strings
- Pointers
- Structures
- File Handling
- Pattern Programs
- Number Programs
- Sorting and Searching
- Mini Programs

## 🛠️ Technologies Used

- C Programming Language
- GCC Compiler
- Visual Studio Code

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/C-Programming.git
```

2. Navigate to the project

```bash
cd C-Programming
```

3. Compile the program

```bash
gcc filename.c -o program
```

4. Run

```bash
./program
```
